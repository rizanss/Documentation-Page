Python Introduction

Apa itu Python?

Python merupakan bahasa pemrograman yang populer. Bahasa Pemrograman ini dibuat oleh Guido van Rossum, dan dirilis pada tahun 1991.

Python bisa digunakan untuk :

  • Web Development (untuk server)
  • Software Development
  • Matematika
  • Machine Learning
  • Data Science

Apa yang bisa Phyton lakukan?

Berikut merupakan beberapa hal yang bisa dilakukan dengan menggunakan bahasa pemrograman Python :

  • Python dapat digunakan di server untuk membuat aplikasi web
  • Python dapat digunakan bersama perangkat lunak untuk membuat alur kerja
  • Python dapat terhubung ke sistem basis data. Itu juga dapat membaca dan memodifikasi file
  • Python dapat digunakan untuk menangani data besar dan melakukan matematika yang rumit
  • Python dapat digunakan untuk pembuatan prototipe cepat, atau untuk pengembangan perangkat lunak siap produksi.

Python Syntax

Mengeksekusi sintaks python

Untuk mengeksekusi sintaks python, langkah paling awal untuk mengenal sintaks dari python adalah sebagai berikut :

print("Hello, World!")

Maka output yang akan muncul setelah mengeksekusi kode diatas adalah

Hello, world!

Phyton Comments

Python memiliki kemampuan berkomentar untuk keperluan dokumentasi dalam kode.

Dimulai dengan #, dan Python akan merender sisa baris sebagai komentar:

#Ini adalah sebuah komentar

print("Hello, World!")

Berdasarkan contoh diatas, baris yang didahulukan dengan tanda "#" tidak akan dieksekusi,

dan hal tersebut (comments) hanya sebagai keperluan dokumentasi.

Python Variables

Variabel

Variabel adalah sebuah entitas yang digunakan untuk menyimpan dan mengakses data atau nilai dalam suatu program.

Variabel memiliki nama yang unik, sehingga Anda bisa mengakses nilai yang disimpan di dalamnya dengan menggunakan nama variabel tersebut.

Membuat Variabel

Python tidak memiliki perintah untuk mendeklarasikan variabel.

Variabel dibuat saat Anda pertama kali memberikan nilai padanya.

Contoh :

x = 5

y = "Joko"

print(x)

print(y)

Berdasarkan kode diatas, "x" dan "y" tersebutlah yang biasa disebut variabel, anda bisa memberi nama variabel sesuka anda.

Asalkan nama variabel tersebut tidak sama dengan nama sintaks yang ada.

Casting

Casting adalah proses konversi tipe data dari satu tipe ke tipe lain.

Dalam bahasa pemrograman Python, casting biasanya dilakukan dengan menggunakan fungsi built-in seperti int(), str(), dan float().

Misalnya, jika Anda ingin mengkonversi bilangan bulat 2 ke tipe string, Anda dapat menggunakan str(2), yang akan mengembalikan string "2".

Contoh :

x = str(3), maka angka 3 tersebut akan berubah menjadi "3"

y = int(3), maka angka tersebut akan menjadi 3

z = float(3), maka angka tersebut akan menjadi 3.0

Python Data Types

Tipe Data

Tipe data adalah kategori data yang menentukan bagaimana data tersebut dapat diproses dan digunakan dalam suatu program.

Dalam bahasa pemrograman Python, beberapa tipe data yang umum digunakan termasuk string, integer, float, dan boolean.

Tipe data string digunakan untuk menyimpan teks, tipe integer digunakan untuk menyimpan bilangan bulat, tipe float digunakan untuk menyimpan bilangan pecahan, dan tipe boolean digunakan untuk menyimpan nilai benar atau salah (true atau false). Penggunaan tipe data yang tepat sangat penting untuk menulis program yang efisien dan mudah dibaca.

Mendapatkan Tipe Data

Anda bisa mendapatkan tipe data dari objek apapun dengan menggunakan fungsi type() :

Contoh :

Cetaklah tipe data dari variabel y

y = 11

print(type(y))

Setelah mengetikkan baris kode diatas, akan muncul output sesuai tipe data yang ada.

Python If Else

Python Condition and If Statements

Python mendukung kondisi logis yang biasa dari matematika :

  • Sama dengan : a == b
  • Tidak sama dengan : a != b
  • Lebih dari : a > b
  • Kurang dari : a < b
  • Lebih dari sama dengan : a >= b
  • Kurang dari sama dengan : a <= b

Pernyataan if dalam Python adalah sebuah pernyataan yang memungkinkan Anda untuk mengeksekusi kode yang berbeda tergantung pada kondisi yang terpenuhi.

Pernyataan if terdiri dari blok kode yang akan dijalankan jika kondisi yang diberikan bernilai True.

contoh :

If statement

a = 20

b = 30

if b > a

print("b lebih besar dari a")

Else

Kata kunci else menangkap apapun yang tidak tertangkap oleh kondisi sebelumnya.

Pernyataan else dalam Python adalah sebuah pernyataan yang digunakan bersama dengan pernyataan if untuk mengeksekusi kode yang berbeda tergantung pada kondisi yang terpenuhi.

Pernyataan else terdiri dari blok kode yang akan dijalankan jika kondisi yang diberikan bernilai False.

Contoh :

Else Statement

a = 20

b = 30

if b > a:

print("b lebih besar dari a")

else

print("a lebih besar dari b")

Elif

Kata kunci elif adalah cara python untuk mengatakan "jika kondisi sebelumnya tidak benar, maka coba kondisi ini".

Contoh :

Elif Statement

a = 10

b = 10

if a > b:

print("a lebih dari b")

elif a == b:

print("a sama dengan b")

Python Operators

Operator digunakan untuk melakukan operasi pada variabel dan nilai.

Python membagi operator dalam grup berikut:

  • Arithmetic Operator (Operator Aritmatika)
  • Assignment Operator
  • Comparison Operator
  • Logical Operator

Operator Aritmatika

Operator aritmatika digunakan dengan nilai numerik untuk melakukan operasi matematika umum:

Berikut adalah contohnya :

  • Pertambahan ("+")

    Contoh :

    x + y

  • Pengurangan ("-")

    Contoh :

    x - y

  • Perkalian ("*")

    Contoh :

    x * y

  • Pembagian ("/")

    Contoh :

    x / y

  • Modulus ("%")

    Contoh :

    x % y

  • Perpangkatan ("**")

    Contoh :

    x ** y

  • Pembagian terdekat (//)

    Contoh :

    x // y

Assignment Operator

Operator penugasan digunakan untuk menetapkan nilai ke variabel:

  • "=", Contoh:

    x = y

  • "+=", Contoh:

    x += y

  • "-=", Contoh:

    x -= y

  • "*-", Contoh

    x *- y

  • "/=", Contoh:

    x /= y

  • "%=", Contoh:

    x %= y

Dan masih banyak operator assignment lainnya, kalian bisa belajar dan mencarinya sendiri.

Comparison Operator

Operator perbandingan digunakan untuk membandingkan dua nilai:

  • Sama dengan (==)

    Contoh :

    x == y

  • Tidak sama dengan (!=)

    Contoh :

    x != y

  • Lebih dari (>)

    Contoh :

    x > y

  • Kurang dari (<)

    Contoh :

    x < y

  • Lebih dari sama dengan (>=)

    Contoh :

    x >= y

  • Kurang dari sama dengan (<=)

    Contoh :

    x <= y

Logical Operator

Operator logika digunakan untuk menggabungkan pernyataan bersyarat:

  • Operator AND

    Contoh :

    x < 5 and x < 10

    Artinya : "Kembalikan nilai True jika kedua pernyataan benar"

  • Operator OR

    Contoh :

    x < 5 or x < 4

    Artinya : "Kembalikan nilai True jika salah satu pernyataan benar"

  • Operator NOT

    Contoh :

    not(x < 5 and x < 10)

    Artinya : "Balikkan hasilnya, kembalikan nilai false jika hasilnya benar"

Python Function

Fungsi adalah blok kode yang hanya berjalan saat dipanggil.

Anda dapat meneruskan data, yang dikenal sebagai parameter, ke dalam suatu fungsi.

Suatu fungsi dapat mengembalikan data sebagai hasilnya.

Membuat sebuah Function

Dalam Python, sebuah fungsi didefinisikan menggunakan kata kunci def:

Contoh :

def my_function():

print("Hello from a function")
